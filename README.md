## What Gets Generated

### Use Case Page Includes:
- **Hero section** with industry-specific background image
- **Challenge section** defining the problem (4-6 pain points)
- **How It Works** with custom SVG process diagram
- **Features section** with 4 capability cards
- **Value metrics** showing measurable results
- **CTA section** with contact button

### Visual Process Diagrams

Choose from 6 diagram types:
1. **Linear Flow** - Simple sequential process
2. **Multi-Input Pipeline** - Data aggregation
3. **Cycle Diagram** - Continuous improvement
4. **Decision Tree** - Branching logic
5. **Hub & Spoke** - Integration platforms
6. **Layered Architecture** - System tiers

Each diagram includes a separator labeled "PROCESS DETAILS" before the detailed text boxes.

## How the Homepage Works

### Automatic Generation

The `update_usecases.py` script:
1. Scans `usecases/` folder
2. Extracts from each HTML file:
   - **Name** from `<meta name="use-case-name">`
   - **Category** from `<meta name="use-case-category">`
   - **Description** from `<meta name="description">`
   - **Slug** from filename
3. Generates `usecases.json`
4. Homepage displays all use cases automatically

### Homepage Features
- **Real-time search** - Type to filter
- **Category filters** - Click to filter by industry
- **Responsive grid** - 3-4 columns ‚Üí 1 column mobile
- **Stats counter** - Shows "4+" use cases

## Required Meta Tags

The template includes 3 meta tags that **must be filled** for homepage display:

```html
<head>
  <title>{{USE_CASE_TITLE}} | INVOKE</title>
  <meta name="use-case-name" content="{{USE_CASE_TITLE}}" />
  <meta name="use-case-category" content="{{USE_CASE_CATEGORY}}" />
  <meta name="description" content="{{USE_CASE_DESCRIPTION}}" />
</head>
```

**Example:**
```html
<meta name="use-case-name" content="Automated Invoice Processing" />
<meta name="use-case-category" content="Finance" />
<meta name="description" content="AI-powered invoice processing that reduces processing time by 90% and eliminates manual data entry errors." />
```

### Categories
Use consistent single-word or two-word categories:
- Supply Chain
- Finance
- Manufacturing
- Healthcare
- Legal
- HR
- Compliance
- Customer Service

## Troubleshooting

**Use case not on homepage?**
```bash
python update_usecases.py
git add usecases.json
git commit -m "Update homepage"
git push
```

**Images broken?**
- Use `../images/` from usecases folder
- Check file exists in images/photos/ or images/icons/

**Fonts not loading?**
- Use `../fonts/font.css` from usecases folder
- Verify font files exist

**Diagram not responsive?**
- Use `viewBox` in SVG, not fixed width/height
